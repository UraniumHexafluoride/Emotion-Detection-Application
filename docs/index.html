Project Overview

Understand the emotions behind text with AI-powered analysis

The Emotion Detection Application is a sophisticated web-based tool that leverages Watson AI's natural language processing capabilities to analyze and identify emotions in text. Going beyond simple sentiment analysis, it can detect nuanced emotions including joy, sadness, anger, fear, and disgust.

Whether you're analyzing customer feedback, social media content, or any text-based communication, this application provides insights into the emotional undertones present in the text, helping you better understand the message behind the words.

Try it out

This is a demonstration. For working analysis, install the application.

Feature Highlights

Text Analysis

Analyze any text input to detect emotional content. The system works with sentences, paragraphs, or longer texts.

Multiple Emotion Detection

Identifies five core emotions (joy, sadness, anger, fear, disgust) with confidence scores for each.

Dominant Emotion

Automatically highlights the strongest emotion present in the analyzed text for quick insights.

Web Interface

User-friendly interface makes it easy to input text and view analysis results in a clean, organized format.

API Access

RESTful API endpoint allows integration with other applications and services for automated emotion analysis.

Error Handling

Robust error handling ensures the application can gracefully manage invalid inputs or service disruptions.

Quick Start Guide

Get Up and Running in 3 Steps
  1. Clone the repository 
    git clone https://github.com/yourusername/emotion-detection-application.git
cd emotion-detection-application
  2. Install dependencies 
    pip install flask requests
  3. Start the application 
    python 2_web_server.py

    Then open your browser and navigate to http://localhost:5000
